MU Connect Heuristic Checklist

Purpose: Quickly identify interface-level issues related to MU Connect’s usability. Please rate each heuristic from 0 (no issue) to 4 (critical issue), and optionally leave a comment.

0 = no issue | 1 = minor issue | 2 = moderate issue | 3 = major issue | 4 = critical issue

1. Visibility of System Status
Does MU Connect keep users informed about what's going on? (e.g., loading indicators, progress bars, status updates on scheduling appointments).
2. Match Between System and the Real World
Are labels, language, icons, and workflows familiar and intuitive, reflecting campus realities (e.g., “Flags,” “Kudos,” “Advisors,” “Appointments”)?
3. User Control and Freedom
Can users easily undo or exit from an action? Are there “Back” or “Cancel” options? Is it straightforward to reverse a mistaken step?
4. Consistency and Standards
Are icons, terminology, and layout consistent across MU Connect pages or sections? Do platform conventions match typical campus expectations?
5. Error Prevention
Does MU Connect proactively prevent user errors (e.g., forms with validation, confirmation dialogs)? Is it hard for users to make mistakes?
6. Recognition Rather Than Recall
Are options, icons, and instructions visible and easy to recognize instead of relying on user memorization? Does the system provide prompts?
7. Flexibility and Efficiency of Use
Can advanced users perform tasks quickly (e.g., keyboard shortcuts, streamlined workflows), while novices still find it accessible?
8. Aesthetic and Minimalist Design
Is the interface visually pleasing and free of irrelevant elements or clutter that might confuse or overwhelm users (especially new students, staff)?
9. Help Users Recognize, Diagnose, and Recover from Errors
When an error occurs (e.g., scheduling conflict, invalid flag entry), does the system provide clear error messages with constructive solutions?
10. Help and Documentation
Does MU Connect provide adequate help pages, tooltips, FAQs, or guidance for both first-time and advanced users? Is it easy to find help resources?
